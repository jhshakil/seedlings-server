# Getting Started

This project is creating with npm package manager

## Install al dependency

run this command
```copy
npm i
```

## Create a .env and add all environment variable

```copy
NODE_ENV=development
PORT=3030
DATABASE_URL=
```

## run project

```copy
npm run start:dev
```

## build project

```copy
npm run build
```

## find eslint error project

```copy
npm run lint
```
